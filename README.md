# 🌙 La Nuit Puff - 線上甜點店舖平台

> 深夜的溫暖陪伴 | 線上購物 + 後台管理系統
> 一個完整的 React + Vite 電商平台，包含前台顧客購物系統與後台管理系統。

---

## ✨ 核心功能

### 👩‍💼 後台管理系統

- **商品管理** - 新增、編輯、刪除、啟用/停用商品
- **訂單管理** - 查看訂單、編輯狀態、刪除訂單
- **文章發佈** - 發表關於我們的文章和新聞
- **圖片上傳** - 支援圖片上傳與 URL 管理

### 🛍️ 前台購物系統

- **商品瀏覽** - 分類篩選、排序、檢視詳情
- **購物車** - 新增商品、調整數量、移除項目
- **結帳系統** - 訂單確認、填入收貨資訊
- **會員功能** - 登入、訂單紀錄、收藏商品
- **品牌介紹** - 關於我們、產品特色、文章閱讀

## 🛠 技術棧

| 層面            | 技術               |
| --------------- | ------------------ |
| **前端框架**    | React 19           |
| **建構工具**    | Vite               |
| **樣式**        | Bootstrap 5 + SCSS |
| **路由**        | React Router v7    |
| **狀態管理**    | Redux Toolkit      |
| **HTTP 客戶端** | Axios              |
| **輪播**        | Swiper             |
| **表單**        | React Hook Form    |
| **提示框**      | SweetAlert2        |
| **動畫**        | AOS 滾動動畫       |

## 📁 專案結構

```
LaNuitPuff/
├── src/
│   ├── components/          # React 組件
│   │   ├── admin/          # 後台組件
│   │   └── ...
│   ├── hooks/               # 自訂 Hooks
│   ├── layout/              # 頁面佈局
│   ├── views/               # 頁面視圖
│   │   ├── admin/           # 後台頁面
│   │   └── frontend/        # 前台頁面
│   ├── services/            # API 服務
│   ├── store/               # Redux 狀態管理
│   ├── assets/              # 靜態資源 & 樣式
│   ├── router.jsx           # 路由設定
│   └── main.jsx             # 應用入口
├── public/                  # 公開資源
└── package.json             # 依賴管理
```

